# Fork 项目特性说明

本项目是 Fluent Reader 的增强版 Fork，重点引入了 AI 辅助阅读能力和性能优化。

## 核心特性

### 1. AI 深度集成
我们将大语言模型（LLM）的能力无缝集成到了 RSS 阅读体验中。

- **智能摘要**:
  - 不再需要阅读长篇大论，AI 会为你提炼文章核心观点。
  - 支持多种模型（GPT-3.5/4, DeepSeek, Claude, 本地 Ollama 等）。
- **无缝翻译**:
  - 独特的"双语对照"模式，保留原文韵味的同时提供流畅的译文。
  - 针对 RSS 文章结构优化的分段翻译算法，阅读体验优于整页翻译插件。
- **划词助手**:
  - 遇到不懂的词句，选中即翻，无需跳出应用。

### 2. 本地化与隐私
我们深知阅读隐私的重要性，因此所有增强功能均遵循"本地优先"原则。

- **SQLite 本地缓存**: 
  - 所有 AI 生成的摘要和翻译都会存储在本地 `ai-cache.db` 中。
  - 同一篇文章永远不会重复调用 API，既省钱又保护隐私。
  - 即使断网，已读过的 AI 内容依然可以访问。
- **自定义 API**:
  - 支持直连 OpenAI，也支持通过代理或本地模型（如 LM Studio, Ollama）使用，数据完全掌握在你手中。

### 3. 效率提升
- **键盘党福音**: 新增了一系列快捷键，你的双手可以不再离开键盘。
  - `Alt+S`: 呼出 AI 摘要
  - `Alt+T`: 开启全文翻译
- **性能调优**: 对文章列表和渲染引擎进行了深度优化，即使订阅数千个源也能流畅滑动。

## 如何开始

1. 下载并安装本 Fork 版本。
2. 进入 **设置 > AI 功能**。
3. 配置你的 API Key 和 Endpoint（支持 OpenAI 格式）。
4. 在阅读文章时，点击工具栏的 AI 图标或使用快捷键体验新功能。
